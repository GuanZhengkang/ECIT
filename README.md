# Efficient Ensemble Conditional Independence Test Framework for Causal Discovery (E-CIT)

We provide a guide to help you get started, please refer to **[tutorial.ipynb](https://github.com/GuanZhengkang/ECIT/blob/main/tutorial.ipynb)**.
